# clinical-trials-db

Учебный проект по SQL и Python для моделирования базы данных клинических исследований.

## Что делает проект

Проект:
- создает структуру базы данных;
- генерирует тестовые данные пациентов, исследований и измерений;
- выполняет аналитические SQL-запросы;
- формирует итоговый CSV-отчет.

## Стек

- Python
- SQLite
- SQL
- SQLAlchemy
- pandas

## Файлы проекта

- `create.sql` — создание таблиц
- `generate_data.py` — генерация данных
- `answers.py` — аналитические SQL-запросы
- `report.sql` — итоговый отчет
- `test.py` — проверка проекта
- `trial_statistics.csv` — пример итогового отчета

## Структура базы данных

База данных состоит из трех таблиц:
- `patients`
- `trials`
- `measurements`

## Как запустить

1. Создать SQLite-базу данных.
2. Выполнить SQL из `create.sql`.
3. Запустить генерацию данных:

```bash
python generate_data.py
